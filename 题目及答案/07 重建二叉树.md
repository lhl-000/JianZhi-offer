### 题目

输入某二叉树的前序遍历和中序遍历的结果，请重建该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。


### 解答

##### 1. 递归 (时间复杂度 O(n))
```
/java/
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        if (preorder == null || inorder == null || preorder.length == 0 || inorder.length == 0) {
            return null;
        }
        TreeNode root = new TreeNode(preorder[0]);
        int index = indexOf(root.val, inorder);
        root.left = buildTree(Arrays.copyOfRange(preorder, 1, index + 1), Arrays.copyOfRange(inorder, 0, index));
        root.right = buildTree(Arrays.copyOfRange(preorder, index + 1, preorder.length), Arrays.copyOfRange(inorder, index + 1,inorder.length));
        return root;
    }

    public int indexOf(int val, int[] a) {
        for (int i=0; i<a.length; i++) {
            if(a[i] == val) {
                return i;
            }
        }
        return -1;
    }
}
```
##### 2. 递归改进版，不用创建子数组，直接在原数组上修改 (时间复杂度 O(n))
```
class Solution {
    private Map<Integer, Integer> indexMap;

    public TreeNode buildTree(int[] preorder, int[] inorder) {
        if (preorder == null || inorder == null || preorder.length == 0 || inorder.length == 0) {
            return null;
        }
        indexMap = new HashMap<>();
        for (int i=0; i<preorder.length; i++) {
            indexMap.put(inorder[i], i);
        }
        return buildTreeNode(preorder, inorder, 0, preorder.length - 1, 0, inorder.length - 1);
    }

    public TreeNode buildTreeNode(int[] preorder, int[] inorder, int pStart, int pEnd, int iStart, int iEnd) {
        if (pStart > pEnd) {
            return null;
        }
        int index = indexMap.get(preorder[pStart]) - iStart;
        TreeNode root = new TreeNode(preorder[pStart]);
        root.left = buildTreeNode(preorder, inorder, pStart + 1, pStart + index, iStart, iStart + index - 1);
        root.right = buildTreeNode(preorder, inorder, pStart + index + 1, pEnd, iStart + index + 1, iEnd);
        return root;
    }
}
/js/
/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {number[]} preorder
 * @param {number[]} inorder
 * @return {TreeNode}
 */
var buildTree = function(preorder, inorder) {
    if (preorder == null || inorder == null || preorder.length == 0 || inorder.length == 0) {
        return null;
    }
    return buildTreeNode(preorder, inorder, 0, preorder.length - 1, 0, inorder.length - 1);

    function buildTreeNode(pre, ino, ps, pe, is, ie){
        if (ps > pe) {
            return null;
        }
        var root = new TreeNode(pre[ps]);
        let index = indexOf(ino, pre[ps]);
        let gap = index - is;
        root.left = buildTreeNode(pre, ino, ps+1, ps + gap , is, index-1);
        root.right = buildTreeNode(pre, ino, ps + gap + 1, pe, index + 1, ie);
        return root;
    };

    function indexOf(array, val) {
        for (let i=0; i<array.length; i++) {
            if (val == array[i]) {
                return i;
            }
        }
    };
};
```
