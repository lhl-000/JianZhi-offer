## 49 丑数

### 题目
我们把只包含质因子 2、3 和 5 的数称作丑数（Ugly Number）。求按从小到大的顺序的第 n 个丑数。

### 解答


##### 1. 优先队列
```
/java/
class Solution {
    public int nthUglyNumber(int n) {
        int[] factors = {2, 3 ,5};
        Set<Long> set = new HashSet<Long>();
        PriorityQueue<Long> pq = new PriorityQueue<Long>();
        int ugly = 0;
        set.add(1L);
        pq.add(1L);
        for (int i=0; i<n; i++) {
            long cur = pq.poll();
            ugly = (int) cur;
            for (long factor : factors) {
                long newUgly = factor * cur;
                if (set.add(newUgly)) {
                    pq.add(newUgly);
                }
            }
        }
        return ugly;
    }
}
```
