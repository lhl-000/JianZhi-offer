## 50 第一个只出现一次的字符

### 题目
在字符串 s 中找出第一个只出现一次的字符。如果没有，返回一个单空格。 s 只包含小写字母

### 解答


##### 1. 哈希表
```
/js/
var firstUniqChar = function(s) {
    let map = new Map();
    for (let i=0; i<s.length; i++) {
        if (map.has(s[i])) {
            map.set(s[i], map.get(s[i])+1);
        } else {
            map.set(s[i], 1);
        }
    }
    for (let v of map.keys()) {
        if (map.get(v) == 1) {
            return v;
        }
    }
    return ' ';
};
```
