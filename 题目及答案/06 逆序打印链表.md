# 05 替换空格

### 题目

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

### 解答

##### 1.递归 (时间复杂度 O(n) 空间复杂度 O(n))

```
/*JAVA*/
class Solution {
    public int[] reversePrint(ListNode head) {
        ArrayList<Integer> arr = new ArrayList<>();
        recursionNode(head,arr);
        int[] res = new int[arr.size()];
        for(int i = 0; i < res.length; i++)
            res[i] = arr.get(i);
        return res;
    }
    public static void recursionNode(ListNode node, ArrayList<Integer> arr) {
        if (node == null) {
            return;
        }
        recursionNode(node.next, arr);
        arr.add(node.val);
    }
}
```
#### 2.直接遍历 (时间复杂度O(n) 空间复杂度为o(1))
```
/java/
class Solution {
    public int[] reversePrint(ListNode head) {
        int cnt = 0;
        ListNode node = head;
        while(node != null) {
            cnt++;
            node = node.next;
        }
        int[] res = new int[cnt];
        node = head;
        for(int i=cnt-1; i>=0; i--) {
            res[i] = node.val;
            node = node.next;
        }
        return res;
    }
}
/js/
function reversePrint(head) {
    var res = new Array;
    let node = head;
    while (node != null) {
        res.unshift(node.val);
        node = node.next;
    }
    return res;
};
```
#### 3.栈 (时间复杂度O(n) 空间复杂度为o(1))
```
/java/
class Solution {
    public int[] reversePrint(ListNode head) {
        Stack<Integer> s = new Stack<>();
        ListNode node = head;
        while(node != null) {
            s.push(node.val);
            node = node.next;
        }
        int num = s.size();
        int[] res = new int[num];
        for (int i=0; i<num; i++) {
            res[i] = s.pop();
        }
        return res;
    }
}
```
