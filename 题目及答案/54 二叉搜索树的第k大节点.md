## 54 二叉搜索树的第k大节点

### 题目
给定一棵二叉搜索树，请找出其中第k大的节点。

### 解答


##### 1. 中序遍历
```
/js/
var kthLargest = function(root, k) {
    let n = 0;
    let res;
    dfs(root)
    return res;
    
    function dfs(node) {
        if (node == null) {
            return;
        }
        dfs(node.right);
        if (++n == k) {
            res = node.val;
        }
        if (n == k) return null;
        dfs(node.left);
    }
};
```
