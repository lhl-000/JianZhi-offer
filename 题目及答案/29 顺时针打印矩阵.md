 # 29  顺时针打印矩阵

### 题目

输入一个矩阵，按照从外向里以顺时针的顺序依次打印出每一个数字。

### 解答


##### 1. 模拟
```
/JAVA/
class Solution {
    public int[] spiralOrder(int[][] matrix) {
        if (matrix == null) {
            return null;
        }
        if(matrix.length == 0) {
            return new int[0];
        }
        int n = matrix.length;
        int m = matrix[0].length;
        int[] res = new int[n*m];
        int top = 0, left = 0, index = 0;
        int right = m-1;
        int btm = n-1;
        while(right >= left && top <= btm) {
            for (int i=left; i<=right; i++) {
                res[index++] = matrix[top][i];
            }
            for (int i=top+1; i <= btm; i++) {
                res[index++] = matrix[i][right];
            }
            if (right > left && top < btm) {
                for (int i=right-1; i>=left; i--) {
                    res[index++] = matrix[btm][i];
                }
                for (int i=btm-1; i>top; i--) {
                    res[index++] = matrix[i][left];
                }
            }
            left++;
            right--;
            top++;
            btm--;
        }
        return res;
     }
}
```
