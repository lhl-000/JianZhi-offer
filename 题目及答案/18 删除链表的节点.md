# 18 删除链表的节点

### 题目

给定单向链表的头指针和一个要删除的节点的值，定义一个函数删除该节点。

返回删除后的链表的头节点



### 解答


##### 1.递归
```
/Java/
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode deleteNode(ListNode head, int val) {
        if (head.val == val ) {
            return head.next;
        }
        head.next = deleteNode(head.next, val);
        return head;
    }
}
```

##### 2.快慢指针
```
/Java/
var deleteNode = function(head, val) {
    if (head.val == val) {
        return head.next;
    }
    let fast = head;
    let slow = head;
    while (fast.val != val) {
        slow = fast;
        fast = fast.next
    }
    slow.next = fast.next;
    return head;
};
