 # 25 二叉树的镜像

### 题目

请完成一个函数，输入一个二叉树，该函数输出它的镜像。

### 解答


##### 1. 二叉树的镜像
```
/JS/
var mirrorTree = function(root) {
    if (root == null) {
        return null;
    }
    let node = new TreeNode(root.val);
    node.val = root.val;
    node.right = node.left = null;
    node.right = mirrorTree(root.left);
    node.left = mirrorTree(root.right);
    return node;
};
/JAVA/
class Solution {
    public TreeNode mirrorTree(TreeNode root) {
        if (root == null) {
            return null;
        }
        return recall(root);
    }

    public TreeNode recall(TreeNode root) {
        if (root == null) {
            return null;
        }
        TreeNode node = new TreeNode(root.val);
        node.left = recall(root.right);
        node.right = recall (root.left);
        return node;
    }
}
```
